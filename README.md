Getting Started
Prerequisites

Ensure you have Python 3.8+ installed.
Installation

    Clone the repository:

git clone https://github.com/eshansharma01/EZ-Assignment-.git
cd EZ-Assignment-

Install dependencies:

pip install -r requirements.txt

Run the application:

    python run.py
